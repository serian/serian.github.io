windows7からwindows10にアップグレードでトラブル

Jul 22, 2016   #win  # 

アップグレード

How to Upgrade to Windows 10 - Microsoftの「無償でアップグレード」ボタンで。

エラーコード0xc1900200

セキュリティーソフトを切らずにアップグレード作業をやってた。
アンチウィルスとアンチリーク機能付きのファイアウォールソフトを停止してクリア。

このプロセッサでは重要な機能 nxがサポートされていません

windows10にしてみた。続。｜西播磨のブログに従って、bcdeditとか言うものでnx値を設定してクリア

インストール後

フォントが汚い

Windows10 フォントが汚いので一発変更! - 窓の杜ライブラリ これ使ってクリア

なんか重い

  • デフォルトのままは危険？ 「Windows 10」のプライバシー設定はこう変えよう ｜ ライフハッカー［日本版］
  • Windows10の音声アシスタント「Cortana」を消す(無効にする) - すなばいじり
  • Windows10はWindows Update時に他ユーザーの帯域を使うことが判明 - GIGAZINE

この辺を見ていろいろオフにしてクリアしつつ。

ダウングレード

１ヶ月以内に戻したくなったら、Windows10 から Windows7/8.1 に戻す方法 - ダウングレード - PC設定のカルマの手順で。

問題点

  • 一瞬、URLショートカットが作れなかったり、通知領域への登録に失敗するソフトが出たりしたけど、再起動で直った。
  • IEの動作が変。
  • やっぱwin7に比べると何割かは余計にメモリ食ってるような。だからwin7 32bitからではあんまりメリットないんじゃないだろうか。
  • outpostが自動的に排除されて、windows defenderに取ってかわられた？

総合

思ったほど重くはない。
